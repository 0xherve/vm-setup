# vm-setup
## Introduction

This repository keeps a bash script to set up a Linux Virtual Machine (VM) after creating it with my essential terminal tools.

## What's included

This script sets up a brand new Ubuntu VM or installation with: 

1. Updated system
2. Essential tools like git, curl, and unzip
3. Install Neovim & Lazy-vim
4.  Install Solana Development tools

> This script is personal and is customized to bring my preferred tools.
